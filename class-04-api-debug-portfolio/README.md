# Class 04: API, Debugging, And Project Explanation

Date: Thursday, May 14, 2026

## Goal

Run a small FastAPI backend on Linux and explain it like a beginner technical project.

This class connects Linux, Python, databases, APIs, debugging, and portfolio communication.

## Start Here

1. Read `LECTURE_NOTES.md`.
2. Complete `LAB.md`.
3. Start with `starter/main.py`.
4. Use `final/main.py` as a reference after trying the lab.

## Final Project

Campus Helpdesk Ticket Tracker:

- `GET /health`
- `GET /tickets`
- `POST /tickets`
- `GET /tickets/{ticket_id}`
- automatic docs at `/docs`
- SQLite persistence

## If Installation Fails

If FastAPI or Uvicorn cannot be installed in your environment, complete the written analysis option in `LAB.md` using the instructor demo, code reading, or a browser environment such as Replit.
