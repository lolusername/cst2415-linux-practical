# Class 03: Python Files And SQLite

Date: Tuesday, May 12, 2026

## Goal

Use Python on Linux to read a CSV file, create a SQLite database, and run simple analysis.

This class moves from command-line analysis to repeatable Python scripts.

## Start Here

1. Read `LECTURE_NOTES.md`.
2. Complete `LAB.md`.
3. Use `data/tickets.csv`.
4. Use the example scripts in `examples/`.

## Why This Matters

Text files are useful, but real applications need persistent data. SQLite is a beginner-friendly database because it stores data in one local file and works with Python without installing a separate database server.
