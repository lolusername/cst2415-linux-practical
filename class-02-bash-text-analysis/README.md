# Class 02: Bash Scripts And Text Analysis

Date: Thursday, May 7, 2026

## Goal

Use Linux command-line tools to search, filter, summarize, and automate work.

This class connects classic Linux course topics to practical job skills:

- Bash scripts
- pipes and redirection
- `grep`
- `find`
- `sort`
- `uniq`
- `wc`
- simple data analysis from text files

## Start Here

1. Read `LECTURE_NOTES.md`.
2. Complete `LAB.md`.
3. Use `data/tickets.csv` as the practice file.
4. Compare your script with `examples/ticket_report.sh` if needed.

## Why This Matters

Support engineers, QA testers, developers, cybersecurity analysts, and cloud technicians often need to inspect logs or exported CSV files quickly. Bash is useful because it lets you combine small commands into practical workflows.
