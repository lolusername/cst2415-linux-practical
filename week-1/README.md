## Class Goal

By the end of this class, you should be able to:

- open a Linux terminal
- move around the filesystem
- create folders and files
- inspect file permissions
- run a Python script
- explain why Linux matters for software development and technical jobs

## Lab Submission

Submit the lab work in Brightspace. Follow the handout exactly.

Your submission should include:

- command evidence or screenshots
- answers to the written questions
- your Python script or its output
- a short reflection on what you learned

Do not submit your work as a GitHub pull request.
